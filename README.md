# Practice-module

## Описание
Дать возможность удобной организации и редактирования учебной дисциплины, желательно на одной веб-странице.
Заполняется преподавателями образовательных проектов Mail.ru.

## Стек технологий
Webpack, preact, redux-zero

## Как запустить  
Сборка:
```
npm i
npm run build
```
